# n8n Automation Notes

GitHub Actions owns most scheduled work to stay within free tiers. n8n Cloud can be used as an optional orchestration layer with five high-value workflows:

1. Weekly executive digest from GitHub workflow artifacts.
2. Manual webhook to trigger content draft generation.
3. Manual webhook to trigger syndication queue review.
4. Alert workflow for failed GitHub Actions.
5. Lead capture notification from ConvertKit or form backend.

Keep secrets inside n8n credentials, not inside this repository.
